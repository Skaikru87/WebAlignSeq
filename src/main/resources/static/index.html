WebAlignSeq

Use this program for aligning your target DNA sequence to reference sequence. The program uses the excel file for taking sequences and storing results. The program puts each nucleotide in separate cell. Additionally, if you provide the data with rs numbers, the program will annotate nucleotides with rs numbers and alleles.
Brief instruction
  • Your excel file should contains reference sequence located in position of c4 (in all sheet which you use).
  • Your target sequences should be located directly below reference sequence (in column C)
  • If you add csv file containing data regarding SNP in your target gene, you should provide starting position of your reference sequence on chromosome (for example: 134886812). This number should be located in position c4 of your excel file.
  • Your csv file should be correctly performed using this instruction.
